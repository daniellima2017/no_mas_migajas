## IMPLEMENTACAO TECNICA (para quando for desenvolver)

### Banco de dados (tabela users - novos campos)
- has_lifetime: boolean (default false)
- has_multimedia: boolean (default false)
- has_narcissist_profile: boolean (default false)
- has_predictor: boolean (default false)
- has_confrontation_sim: boolean (default false)

### Webhook Hotmart
- Cada order bump eh um produto separado na Hotmart
- Webhook recebe o ID do produto e ativa o campo correspondente

### No app
- Botao dourado = ativo (campo true)
- Botao cinza com cadeado = inativo (campo false)
- Click no cinza = abre URL do checkout Hotmart do order bump especifico

### Mensagens contextuais
- Cron job ou trigger baseado em uso (entradas no diario, uso do simulador, dias de streak)
- Maximo 1 mensagem contextual por dia para nao irritar
- Sempre conectada com algo que ela FEZ, nunca aleatoria
